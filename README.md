# Music for Reading / Музыка для чтения

Офлайн Android-приложение: процедурная инструментальная музыка для чтения с маскировкой внешнего шума (Oboe + C++).

## Скачать APK

Готовый файл для установки: **[releases/MusicForReading.apk](releases/MusicForReading.apk)**

1. Скачайте APK на телефон
2. Разрешите установку из неизвестных источников (если система попросит)
3. Откройте файл и установите

Требования: Android 8.0+ (API 26).

## Возможности

- 6 стилей фоновой музыки без вокала и ударных
- Постоянный шумовой слой для маскировки транспорта и разговоров
- Полностью офлайн — без доступа в интернет
- Бесконечное процедурное воспроизведение
- Фоновый плеер с уведомлением
- Таймер сна (15 / 30 / 60 / 90 мин)
- Интерфейс на русском и английском (Material Design 3)

## Сборка из исходников

Нужны: Android Studio Ladybug (2024.2+) или новее, SDK 35, NDK 27+, JDK 17.

```bash
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`

Release-бандл для Play Store:

```bash
./gradlew bundleRelease
```

## Структура проекта

- `app/src/main/java/` — Kotlin UI (Jetpack Compose) и сервис воспроизведения
- `app/src/main/cpp/` — аудиодвижок Oboe и процедурный синтез
- `releases/` — готовый APK для установки
- `docs/` — материалы для публикации в Play Store
